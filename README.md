# Data Structures & Algorithms written in **Python**

### Overview

- **Arrays**
    - Static
        - lookup O(1)
        - push O(1)
        - insert O(n)
        - delete O(n)
    - Dynamic
        - lookup O(1)
        - append O(1)
            - can be O(n) on expanding memory
        - insert O(n)
        - delete O(n)
- **Hash Tables (Dictionary)**
    - space O(n)
    - insert O(1)
    - lookup O(1)
    - delete O(1)
- **Linked Lists**
    - prepend O(1)
    - append O(1)
    - lookup O(n)
    - insert O(n)
    - delete O(n)
- **Stacks**
    - lookup O(n)
    - pop O(1)
    - push O(1)
    - peek O(1)
- **Queues**
    - lookup O(n)
    - enqueue O(1)
    - dequeue O(1)
    - peek O(1)
